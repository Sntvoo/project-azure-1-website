Key Features

Gemini AI

Full access to Gemini 2.5 Flash model — chat freely about anything. Most safety restrictions removed (except illegal/harmful topics like bomb-making or extreme explicit content).

WHOIS Suite

Custom-built WHOIS tool: enter any domain and get full details instantly in the integrated command-line interface.

The Vault

Encrypt/decrypt any file with AES-256. Set custom passwords and manage files directly in-app.

Intelligence Links

One-click access to VirusTotal, HaveIBeenPwned, Shodan, and more — opens in your default browser.

Game Server Status

Live Rainbow Six Siege server status across all platforms, with raw HTML view and direct source link.

Gaming Hub

Latest headlines from PC Gamer, IGN, Kotaku, and GameSpot — organized in tabs, click to read full articles.

System Defence

Enhanced Task Manager: view processes, detect suspicious activity, and monitor priority levels.

Network Scanning

Full port scanning, ICMP ping/traceroute, and network diagnostics — all built-in.
